Topics that we are going to cover in 60 Days

Phase Days Topics Key Activities
Phase 1: Core Web Development Day 1–3 HTML Basics Learn HTML structure, semantic tags, and forms. Build a basic webpage.
Day 4–7 CSS Basics Master Flexbox, Grid, and Responsive Design. Style your HTML webpage.
Day 8–10 Git & GitHub Learn version control basics. Create and manage repositories on GitHub.
Week 3–4 Day 11–14 JavaScript Fundamentals Variables, functions, loops, and closures. Practice coding exercises.
Day 15–17 DOM Manipulation Interact with the DOM, Event Listeners, and dynamic UI changes.
Day 18–20 Advanced JavaScript Promises, Async/Await, Error Handling. Build a small interactive project.
Week 5–6 Day 21–25 React Basics JSX, Components, Props, State. Build simple components.
Day 26–28 Advanced React Hooks (useState, useEffect), React Router. Create functional components.
Day 29–30 React Project Build a small React App (e.g., To-Do List, Weather App).
Phase 1.5: Next.js Integration Day 31–33 Next.js Basics Pages, Routing, API Routes, Static Site Generation (SSG).
Day 34–35 Next.js Advanced Server-Side Rendering (SSR), Dynamic Routing, Middleware.
Day 36–37 Next.js Project Build a dynamic Next.js project (e.g., Blog App, Portfolio).
Phase 2: Backend Development Day 38–41 Node.js Basics Setup Node.js, HTTP Module, Build a server.
Day 42–45 Express.js Basics REST APIs, Middleware, Routing. Build an API server.
Day 46–48 Database Integration MySQL/MongoDB CRUD operations. Secure APIs with JWT.
Phase 2.5: Full-Stack Integration + Deployment Day 49–51 Full-Stack Integration Connect Next.js Frontend with Node.js Backend.
Day 52–53 Authentication Implement JWT-based Authentication and Secure Routes.
Day 54 Deployment Deploy frontend (Vercel) & backend (Render/AWS).
Capstone Project Day 55–56 Build a Full-Stack Project Build an E-commerce or Social Media Platform using Next.js & Node.js.
Phase 3: Interview Preparation Day 57–58 DSA Basics Arrays, Strings, HashMaps, Stacks, Queues. Practice on LeetCode.
Day 59–60 System Design High-Level Architecture, Database Design, API Principles.
Day 61–62 Behavioral Interviews STAR Method, HR Questions, Craft answers for common scenarios.
Day 63–64 Mock Interviews Practice technical and behavioral mock interviews.
Day 65 Final Polish Optimize resume, refine LinkedIn profile, and prepare an elevator pitch.
